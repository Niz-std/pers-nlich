Keywords

Berikut adalah Keywords yang wajib dipahami dan dihafal agar kemudian mempermudah didalam memelajari bahasa C++

    • Compilers : Penerjemah High Level language etc, kedalam bahasa mesin (Biner)


      Jenis-jenis mesin Compilers :

        1. GCC


        2. Clang


      Console Programs : Program yang menggunakan text untuk berkomunikasi dengan user dan environment seperti printing text ke layar atau membaca input dari keyboard.

      IDE : yakni (Integrated Development Environment) adalah sebuah alat pengompilasi bahasa yang sudah terdiri dari beragam fitur pengembang termasuk Text editor dan alat (mesin) untuk mengompilasi program secara langsung


      Jenis-jenis IDE yang kerap digunakan :


        1. Code::blocks


        2. Visual Studio Express


        3. Dev-C++ (Obsolete)


      Preprocessor : Baris spesial yang dinterpretasikan sebelum kompilasi program tersebut dimulai.

      Variables : adalah bagian/porsi dari memori (RAM) tuk menyimpan suatu nilai

      Identifier : adalah pengidentifikasi suatu variabel, atau nama variabel


        Valid Identifier : adalah rangkaian yang terdiri dari 1 atau lebih huruf, digit, karakter underscore(_).

        Standard reserved keywords yang tidak bisa digunakan oleh programmer.
        alignas, alignof, and, and_eq, asm, auto, bitand, bitor, bool, break, case, catch, char, char16_t, char32_t, class, compl, const, constexpr, const_cast, continue, decltype, default, delete, do, double, dynamic_cast, else, enum, explicit, export, extern, false, float, for, friend, goto, if, inline, int, long, mutable, namespace, new, noexcept, not, not_eq, nullptr, operator, or, or_eq, private, protected, public, register, reinterpret_cast, return, short, signed, sizeof, static, static_assert, static_cast, struct, switch, template, this, thread_local, throw, true, try, typedef, typeid, typename, union, unsigned, using, virtual, void, volatile, wchar_t, while, xor, xor_eq
